# CS 303 Project: Password Strength Checker
### A password is considered strong if it meets the following criteria: 

    Has at least one uppercase English character
    Has at least one lowercase English character
    Has at least one of the following special characters: !@#$%^&*()-+_=
    Has at least one number
    Has a length of at least 8 characters

A password is strong if all above conditions are met, moderate if the first three conditions are met and has a length of at least 6 characters, otherwise it is considered weak
